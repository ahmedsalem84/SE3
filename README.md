# MicroService-CSA: LLM-Driven Semantic Embeddings & Circle Search Optimization

MicroService-CSA is an automated code refactoring framework designed to decompose monolithic Java applications into microservices. It bridges the "Fitness-Fidelity Gap" by combining the geometric exploration capabilities of the **Circle Search Algorithm (CSA)** with context-aware semantic vector representations generated by **Large Language Models (LLMs)**.

This repository contains the complete implementation, datasets, evaluation pipeline, and results of the comparative study against seven state-of-the-art metaheuristics.

---

## Key Features

1. **Dual-Channel Data Extraction**: Extracts structural dependencies (method calls, field accesses, inheritance) via AST parsing and builds a weighted dependency graph.
2. **LLM-Based Semantic Embeddings**: Cleans and ingests class-level source code, generating 1024-dimensional semantic vectors using locally hosted LLMs via Ollama.
3. **Circle Search Algorithm (CSA)**: Utilizes a geometric concentric search mechanism to navigate the NP-hard clustering space without getting trapped in local optima.
4. **Hybrid Multi-Objective Fitness Function**: Balances structural modularity (TurboMQ) with semantic cohesion (Cosine Similarity of embeddings).
5. **LLM-Driven Service Naming**: Automatically aggregates class context within resolved clusters to propose domain-relevant microservice names.


Getting Started

Prerequisites

  - Python: version 3.9 or higher
  - Ollama: install locally to run the embedding model. Ensure Ollama is running
    in the background.

Setup Instructions

1.  Clone the repository:

    git clone https://github.com/ahmedsalem84/SE3.git
    cd SE3

2.  Install dependencies:

    pip install -r requirements.txt

3.  Pull the Llama3 model:

    ollama pull llama3

Running the Experiments

1. Data Processing and Embedding Generation

Extract class structural graphs and semantic embeddings for a target dataset:

python src/main.py --dataset PetClinic --phase preprocess

2. Multi-Objective Optimization

To run the MicroService-CSA framework (or other baselines like PSO, GA, etc.) on
a selected dataset:

python src/main.py --dataset PetClinic --optimizer CSA --runs 30

3. Hyperparameter Tuning (Optuna)

Run the Optuna study to find the optimal trade-off weights (w_1, w_2):

python src/evaluation/tune_hyperparameters.py --dataset PetClinic
